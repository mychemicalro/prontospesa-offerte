# 🛒 Pronto Spesa — Offerte

Sito web con le migliori offerte **PEWEX/Pronto Spesa**, aggiornato automaticamente ogni giorno.

## Come funziona

- Ogni mattina alle **07:00** i prezzi vengono scaricati
- Alle **07:20** i dati vengono pubblicati su questo sito
- **924+ offerte** monitorate quotidianamente

## Tecnologia

- Sito statico HTML/CSS/JS su **GitHub Pages**
- Dati in JSON aggiornati automaticamente da **Hermes Agent**
- Fonte: [PEWEX](https://pewex.it)
